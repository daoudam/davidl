    Evernote

    Evernote est un moyen rapide et facile des synchroniser du contenu à travers toutes vos machines. Si vous créez un compte à partir du lien suivant, cela vous donne le droit à un mois d'abonnement premium gratuitement

    Visiter le site

    Firefox

    Le navigateur web sécurisé, performant et personnalisable.

    Visiter le site

    Firefox Nightly

    La version alpha de Firefox permet de tester en avant-première les futures publications du navigateur.

    Visiter le site

    GitHub

    Hébergement de vos projets utilisant Git. Le site web possède des fonctionalités indispensable comme un navigateur de code source, l'édition en ligne, les wikis, et une gestion de tickets.

    Visiter le site

    GitX

    GitX est une interface graphique de git fait pour Mac OS X. Il dispose actuellement d'un historique comme gitk et une interface graphique de commit comme git gui. Mais il conserve le style Mac OS X !

    Visiter le site

    ImageOptim

    ImageOptim optimise les images PNG, JPEG et GIF animés. Elles prennent moins de place sur votre disque et se chargeront plus vite sur votre site web.

    Visiter le site

    ImageAlpha

    ImageAlpha réduit considérablement la taille des fichiers PNG24 bits (y compris la transparence alpha) en appliquant une compression avec perte + une conversion PNG8 plus efficace et y ajoute le format alpha.

    Visiter le site

    iTerm2

    iTerm2 est un émulateur de terminal pour Mac OS X qui fait des choses étonnantes.

    Visiter le site

    JPEGmini

    Avec JPEGmini pour Mac, vous pouvez réduire la taille du fichier de vos photos jusqu'à 5x, tout en gardant leur qualité d'origine et le format JPEG.

    Visiter le site

    LICEcap

    LICEcap permet d'enregistrer une partie de votre écran pour la convertir en .gif animé. Simple et efficace.

    Visiter le site

    Papier et crayon

    Si vous ne possédez pas de carnet de notes et un beau crayon, s'il vous plaît faites-vous une faveur, arrêter de lire ce site et allez en acheter un. Forcez-vous à l'utiliser. Ensuite, remarquez comment il est plus facile de poser et d'articuler ses idées de design avec cet outil.

    Visiter le site

    PixelWindow

    Une application simple et multi-plateforme pour mesurer des choses sur votre écran.

    Visiter le site

    Rename

    Renomer ses fichiers, de la façon la plus simple possible.

    Visiter le site

    Screaming Frog SEO Spider

    Ce logiciel permet d'analyser et d'effectuer rapidement l'examen comple d'un site d'un point de vue SEO. Il es particulièrement efficace pour les sites avec des milliers d'url où il est impossible de tout vérifier à la main.

    Visiter le site

    Sequel Pro

    Sequel Pro est une application rapide & facile à utiliser pour travailler avec des bases de données MySQL.

    Visiter le site

    Sublime Text

    Selon moi c'est le meilleur éditeur de code. Facile à prendre en main, et évolutif presque à l'infini grâce à une énorme quantité de plug-in.

    Visiter le site

    Tower

    Git tower un un bon remplaçant à GitX. Il dispose de beaucoups plus de fonctionalités et pourra vous sauver la vie pendant la résolution de conflits sous git.
